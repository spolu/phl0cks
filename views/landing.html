Phl0cks

Phl0cks is a programming game in which two or more programs control spaceships with the goal of eliminating every other enemy ships and survive as long as possible. Known as `phl0cks`, the programs are written in JavaScript and submitted by the players to compete against each others.

Watch a Phl0cks Match

Building the perfect phl0ck will require studying complex machine learning algorithms and IA techniques, but you can start from the beggining and progress with your friends, combatting their phl0cks and improving yours through challenges sent to each others.

Getting Started [adventurous geek mode]

sudo npm install phl0cks -g

phl0cks help

Getting Started [beginner mode]

  • Get ready to tackle difficulties
  • But it's a lot of fun and maybe on of the best way to learn JavaScript!
  • First, Install NodeJS [go here]
  • Then, Open a Terminal (`Terminal` on OSX, `cmd` on Windows, `xterm` on Linux)
    Google can help you here...
  • Install Phl0cks

    sudo npm install phl0cks -g

  • Signup for phl0cks

    phl0cks signup

  • Verify your accout [you should have received an email] (replace <...> by actual values)

    phl0cks verify <code>

  • Create a dummy phl0ck [see this]

    phl0cks boilerplate dummy > dummy.js

  • Simulate a combat between 2 instances of your phl0ck [see this again]

    phl0cks simulate 4 dummy.js dummy.js | phl0cks play

  • Accept the challenge you received

    phl0cks challenge accept <id> <code>

  • Submit your dummy phl0ck to combat for the challenge

    phl0cks challenge submit <id> dummy.js

  • You're all set!
    Edit dummy.js and make it more powerful to win your challenges!
